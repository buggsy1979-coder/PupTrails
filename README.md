# PupTrails
A full Animal rescue and adoption package with all the bells and whistles.

PupTrails is a Windows WPF desktop application designed for dog rescue and shelter operations. It brings intake, medical history, adoptions, financial tracking, and reporting into one workflow with a portable, local SQLite data store.

This repository hosts release downloads only. Source code is not published.

Downloads:
- Go to the Releases page and download the latest build: https://github.com/buggsy1979-coder/PupTrails/releases

## Features
- Animal intake and profile management with status tracking
- People records (adopters, vets, contacts, volunteers)
- Adoptions with fees, contracts, and payment status
- Adoption appointments window with action buttons and dialogs
- Vet visits with services, procedures, and vaccination tracking
- Transport trips with animal assignments and costs
- Expenses, income, and money owed tracking
- Puppy groups and group photo support
- Attachments for photos, receipts, invoices, and contracts
- Advanced search and filtering
- CSV and social media export support
- Automatic backups and diagnostics logging
- Portable storage layout for data and attachments

## System Requirements
- Windows 10 or newer
- .NET 8 Desktop Runtime

## Download and Run
1. Go to the Releases page and download the latest `PupTrails-win-x64.zip`.
2. Extract the zip file. It will create a `PupTrails` folder.
3. Keep the `PupTrails` folder in the root location you plan to use.
4. Run `PupTrails.exe` from inside the `PupTrails` folder.

## Data Storage and Portability
PupTrails is fully portable. Keep the application inside a single root folder named `PupTrails`.
All application data is stored under a portable folder next to the app binaries:

`PupTrails\PupTrailsDocs\`

Key folders:
- `PupTrailsDocs\data` (SQLite database)
- `PupTrailsDocs\attachments` (photos, receipts, contracts)
- `PupTrailsDocs\backups` (automatic/manual backups)
- `PupTrailsDocs\logs` (diagnostic logs)

The app will create these folders on first run if they do not exist.
